Changing HTML

JavaScript allows us to create HTML dynamically. This means that we can change (create, read, update, and destroy) the HTML elements and their contents.

Changing the HTML Output Stream

By using the document.write() method, we can directly write to the HTML output stream.

Warning! Never use document.write() after the document is loaded. This will overwrite the entire document.

Example

The following example will overwrite the entire document with the current datetime. 

<!DOCTYPE html>
<html>
<body>

<script>
document.write(Date());
</script>

</body>
</html>

See the example in action.

Changing HTML Content

Using innerHTML is the easiest way to change the content of an HTML element.

Heads up! Unlike document.write(), this will not change any of the contents of the other HTML elements in the document. It will only change the contents of the element that you specify.

Example

The following example will change the content of the p tag that has the id of "p1" from "Hello, World!" to "New text!" 

<html>
<body>

<p id="p1">Hello, World!</p>

<script>
document.getElementById('p1').innerHTML = "New text!";
</script>

</body>
</html>

See the example in action.

Changing the Value of an Attribute

The attributes (such as src and href) can be changed dynamically by using the following syntax 

document.getElementById(id).attribute = new value;

where attribute is the attribute you wish to change.

Example

Here, we're going to change the src attribute of an img element from one image to another. 

<html>
<body>

<img id="myImage" src="smiley.gif"/>

<script>
document.getElementById("myImage").src = "landscape.jpg";
</script>

</body>
</html>

See the example in action.
