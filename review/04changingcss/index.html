Changing CSS

The DOM API allows JavaScript to change the CSS styling of HTML elements.

Changing HTML Style

The syntax to change the styling of HTML elements is as follows 

document.getElementById(id).style.color = new style;

where the id is the id of the element and new style is the new value of the style property.

Example

The following example will change the font color of the element with the id of p2 to blue. 

<html>
<body>

<p id="p2">Hello, World!<p/>

<script>
document.getElementById("p2").style.color = "blue";
</script>

</body>
</html>

Using Events

The DOM allows code to be executed when certain events occur. Examples of these are (but no limited to)

  • an element is clicked on
  • the page has loaded
  • input fields have changed

Example

The following example changes the font color of the element with the id of "id1" from black to red when the button is clicked. 

<html>
<body>

<p id="id1">My Heading 1<p/>

<button type="button"
        onclick="document.getElementById('id1').style.color = 'red'">
  Click Me!
</button>

</body>
</html>
