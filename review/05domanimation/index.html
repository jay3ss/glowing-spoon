HTML DOM Animation

Now, we're going to learn how to create HTML animations using JavaScript.

A Basic Web Page

Before we learn to create HTML animations, we'll need a basic web page to animate. This page is shown below. 

<!DOCTYPE html>
<html lang="en">
<body>

<h1>My First JavaScript Animation</h1>

<div id"animation">My First JavaScript Animation</div>

</body>
</html>

Create the Animation Container

We're going to create a container that all animations will be relative to. 

<div id="container">
  <div id="animate">My animation will go here</div>
</div>

Styling the Elements

We're going to style the "container" element with position: relative and the "animation" element with position: absolute. We'll also size and style them, as shown below 

#container {
  width: 400px;
  height: 400px;
  position: relative;
  background: yellow;
}

#animate {
  width: 50px;
  height: 50px;
  position: absolute;
  background: red;
}

Animation Code

The animations are achieved by programming gradual changes in an element's style. A timer with a small enough interval calls these changes to make them appear to be continuous. The basic code is given below 

var id = setInterval(frame, 5);

function frame() {
  if (/* test for finished */) {
    clearInterval(id);
  }
  else {
    /* code to change the element's style */
  }
}

Creating the Animation with JavaScript

The code below will move the element with the id of "animate" down and to the right until the pos variable has a value of 350. 

function myMove() {
  var elem = document.getElementById("animate");
  var pos = 0;
  var id = setInterval(frame, 5);

  function frame() {
    if (pos == 350) {
      clearInterval(id);
    } else {
      pos++;
      elem.style.top = pos + 'px';
      elem.style.left = pos + 'px';
    }
  }
}

Creating an Event

Now that we have the code to animate an element, we need a way to start it. To do that, we'll use a button as shown below.

Example

The code below shows a button that, when clicked, class the myMove function which will begin the animation. 

<button type="button"
        onclick="myMove();">
  Click here
</button>

Putting it All Together

Now, that we've gotten every part down, let's combine them to the final product as shown below. 

<!DOCTYPE html>
<html lang="en">
<head>
  <style>
    #container {
      width: 400px;
      height: 400px;
      position: relative;
      background: yellow;
    }

    #animate {
      width: 50px;
      height: 50px;
      position: absolute;
      background: red;
    }
  </style>
</head>
<body>

  <div id="container">
    <div id="animate"></div>
  </div>

  <button type="button" onclick="myMove();">
    Click here
  </button>

  <script>
    function myMove() {
      var elem = document.getElementById("animate");
      var pos = 0;
      var id = setInterval(frame, 5);

      function frame() {
        if (pos == 350) {
          clearInterval(id);
        } else {
          pos++;
          elem.style.top = pos + 'px';
          elem.style.left = pos + 'px';
        }
      }
    }
  </script>

</body>
</html>
